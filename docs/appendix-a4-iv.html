Appendix A4. Instrumental Variables (IV / 2SLS) Example

  • Show All Code
  • Hide All Code
  • View Source

Overview

This appendix gives a small IV example using simulated data.
We generate a setting where the treatment (friends, having friends in the classroom) is confounded by an unobserved factor (motivation).
A plausible instrument (distance, minutes to school) affects the treatment but does not directly affect test scores except through treatment.

We then estimate the effect of friends on scores using 2SLS and run a konfound sensitivity analysis on the IV estimate.

A4.1 Setup

Code 
set.seed(123)

library(AER)      # provides ivreg()
library(konfound)
library(dplyr)

A4.2 Simulate data

A quick note on the simulation logic:

  • motivation affects both friends and scores (this is the confounder).
  • distance shifts the likelihood of friends but does not enter the outcome equation directly.
Code 
n <- 2000

# Instrument: distance to school (minutes)
distance <- rnorm(n, mean = 20, sd = 8)

# Unobserved confounder: student motivation
motivation <- rnorm(n)

# Treatment: having friends in the classroom (binary)
prob_friends <- plogis(-0.5 + 0.08 * distance - 0.5 * motivation)
friends <- rbinom(n, 1, prob_friends)

# Outcome: test scores (friends has a true causal effect of +3)
scores <- 50 + 3 * friends + 5 * motivation + rnorm(n, sd = 5)

dat <- data.frame(scores, friends, distance, motivation)

summary(dat)

A4.6 Sensitivity analysis (Konfound)

Below we run konfound using the estimate and SE you provided.

If you want this fully reproducible end-to-end, we can extract these values directly from the model object instead of hard-coding them.

Code 
pkonfound(
  est_eff      = 2.5918,
  std_err      = 1.2526,
  n_obs        = 2000,
  n_covariates = 2,
  index        = "RIR"
)
Robustness of Inference to Replacement (RIR):
RIR = 104

To nullify the inference of an effect using the threshold of 2.457 for
statistical significance (with null hypothesis = 0 and alpha = 0.05), 5.219%
of the estimate of 2.592 would have to be due to bias. This implies that to
nullify the inference one would expect to have to replace 104 (5.219%)
observations with data points for which the effect is 0 (RIR = 104).

See Frank et al. (2013) for a description of the method.

Citation: Frank, K.A., Maroulis, S., Duong, M., and Kelcey, B. (2013).
What would it take to change an inference?
Using Rubin's causal model to interpret the robustness of causal inferences.
Education, Evaluation and Policy Analysis, 35 437-460.

Accuracy of results increases with the number of decimals reported.
Code 
pkonfound(
  est_eff      = 2.5918,
  std_err      = 1.2526,
  n_obs        = 2000,
  n_covariates = 2,
  index        = "IT"
)
Impact Threshold for a Confounding Variable (ITCV):
The minimum impact of an omitted variable to nullify an inference for
a null hypothesis of an effect of 0 (nu) is based on a correlation of 0.05
with the outcome and 0.05 with the predictor of interest (conditioning on all
observed covariates in the model; signs are interchangeable if they are different).
This is based on a threshold effect of 0.044 for statistical significance (alpha = 0.05).

Correspondingly the conditional impact of an omitted variable (as defined in Frank 2000) must be 
0.05 X 0.05 = 0.003 to nullify an inference for a null hypothesis of an effect of 0 (nu).

For calculation of unconditional ITCV using pkonfound(), additionally include
the R2, sdx, and sdy as input.

See Frank (2000) for a description of the method.

Citation:
Frank, K. (2000). Impact of a confounding variable on the inference of a
regression coefficient. Sociological Methods and Research, 29 (2), 147-194

Accuracy of results increases with the number of decimals reported.

The ITCV analysis was originally derived for OLS standard errors. If the
standard errors reported in the table were not based on OLS, some caution
should be used to interpret the ITCV.

## Overview

This appendix gives a small IV example using simulated data.  
We generate a setting where the treatment (`friends`, having friends in the classroom) is confounded by an unobserved factor (`motivation`).  
A plausible instrument (`distance`, minutes to school) affects the treatment but does not directly affect test scores except through treatment.

We then estimate the effect of `friends` on `scores` using 2SLS and run a [konfound](https://cran.r-project.org/web/packages/konfound/index.html) sensitivity analysis on the IV estimate.

---

## A4.1 Setup

```{r}
#| label: a4-setup
#| include: true

set.seed(123)

library(AER)      # provides ivreg()
library(konfound)
library(dplyr)
```

## A4.2 Simulate data

A quick note on the simulation logic:

- `motivation` affects both `friends` and `scores` (this is the confounder).
- `distance` shifts the likelihood of `friends` but does not enter the outcome equation directly.

```{r}
#| label: a4-simulate
#| include: true

n <- 2000

# Instrument: distance to school (minutes)
distance <- rnorm(n, mean = 20, sd = 8)

# Unobserved confounder: student motivation
motivation <- rnorm(n)

# Treatment: having friends in the classroom (binary)
prob_friends <- plogis(-0.5 + 0.08 * distance - 0.5 * motivation)
friends <- rbinom(n, 1, prob_friends)

# Outcome: test scores (friends has a true causal effect of +3)
scores <- 50 + 3 * friends + 5 * motivation + rnorm(n, sd = 5)

dat <- data.frame(scores, friends, distance, motivation)

summary(dat)
```

## A4.3 Why IV is useful here (quick check)

If we regress `scores` on `friends` directly, motivation will bias the estimate because it’s omitted and correlated with `friends`.

```{r}
#| label: a4-ols
#| include: true

ols_model <- lm(scores ~ friends, data = dat)
summary(ols_model)
```



## A4.4 2SLS / IV estimation

We instrument `friends` with `distance`:

[

scores = \beta_0 + \beta_1 friends + \varepsilon,\quad friends \leftarrow distance

]

```{r}
#| label: a4-iv
#| include: true

iv_model <- ivreg(scores ~ friends | distance, data = dat)
summary(iv_model)
```


## A4.5 First-stage strength (diagnostic)

This is a practical check: if the instrument barely moves treatment, IV will be noisy and potentially misleading.

`summary(iv_model, diagnostics = TRUE)` reports common diagnostics (including a weak instrument test).

```{r}
#| label: a4-iv-diagnostics
#| include: true

summary(iv_model, diagnostics = TRUE)
```
