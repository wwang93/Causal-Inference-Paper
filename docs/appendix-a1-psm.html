Appendix A1. Propensity Score Weighting (CBPS-IPTW)

  • Show All Code
  • Hide All Code
  • View Source

Overview

This appendix example estimates the average treatment effect (ATE) of having an IEP on science self-efficacy (X1SCIEFF). We (1) build an analytic dataset, (2) estimate propensity score weights using CBPS, (3) check covariate balance, (4) fit a weighted outcome model, and (5) run a sensitivity analysis with konfound.

Code 
set.seed(12345)

# Packages used in this example
library(dplyr)

library(twang)
library(survey)
library(konfound)

A1.5 Construct analysis variables (treatment + covariates)

The goal here is not to be clever—just to turn a handful of key survey fields into clean indicators that are easy to use in the weighting model.

Code 
analytic <- vars_clean %>%
  mutate(
    # treatment: IEP
    IEP = if_else(X1IEPFLAG == "Student has an IEP", 1, 0, missing = NA),

    # demographics
    female = if_else(X1SEX == "Female", 1, 0, missing = NA),

    race_white = if_else(X1RACE == "White, non-Hispanic", 1, 0, missing = NA),
    race_black = if_else(X1RACE == "Black/African-American, non-Hispanic", 1, 0, missing = NA),
    race_asian = if_else(X1RACE == " Asian, non-Hispanic ", 1, 0, missing = NA),
    race_hispanic = if_else(X1RACE == " Hispanic, race specified ", 1, 0, missing = NA),

    # language / locale
    dual_lang = if_else(X1DUALLANG == "First language is English only", 1, 0, missing = NA),
    locale_rural = if_else(X1LOCALE == "Rural", 1, 0, missing = NA),

    # family background
    divorced = if_else(P1MARSTAT %in% c("Divorced", "Separated"), 1, 0, missing = NA),
    bachelorabove1 = if_else(P1HIDEG1 %in% c("Bachelor's degree", "Master's degree"), 1, 0, missing = NA),
    bachelorabove2 = if_else(P1HIDEG2 %in% c("Bachelor's degree", "Master's degree"), 1, 0, missing = NA),

    # parent-reported conditions
    LD = if_else(P1SLD == "Yes", 1, 0, missing = NA),
    DD = if_else(P1DD == "Yes", 1, 0, missing = NA)
  ) %>%
  mutate(
    # SES: convert ordered quintiles into a simple numeric scale 1–5
    SES = factor(
      X1SESQ5,
      levels = c(
        "First quintile (lowest)",
        "Second quintile",
        "Third quintile",
        "Fourth quintile",
        "Fifth quintile (highest)"
      ),
      labels = c(1, 2, 3, 4, 5)
    ) %>% as.numeric()
  )

A1.7 Estimate CBPS weights (ATE)

We use CBPS to estimate weights that balance covariates by construction.

Then we normalize the weights so the average weight is 1 (helps with interpretability and avoids tiny/huge totals).

Code 
cbps_fit <- CBPS(
  IEP ~ female + race_white + race_black +
    dual_lang + locale_rural + SES + divorced +
    bachelorabove1 + bachelorabove2 + LD + DD,
  data = analytic_clean,
  method = "exact",
  ATT = 0
)

analytic_clean$IPTW <- cbps_fit$weights

balance_before <- bal.stat(
  data = analytic_clean,

A1.10 Sensitivity analysis (Konfound)

Here we run konfound using the reported estimate and standard error.

In practice, you’ll usually pull the estimate/SE directly from the fitted model output, but the hard-coded values are fine for a fixed appendix example.

Code 
# RIR index
pkonfound(
  est_eff = -0.63613,
  std_err = 0.16460,
  n_obs = 5282,
  n_covariates = 1,
  index = "RIR"
)
Robustness of Inference to Replacement (RIR):
RIR = 2603

To nullify the inference of an effect using the threshold of -0.323 for
statistical significance (with null hypothesis = 0 and alpha = 0.05), 49.274%
of the estimate of -0.636 would have to be due to bias. This implies that to
nullify the inference one would expect to have to replace 2603 (49.274%)
observations with data points for which the effect is 0 (RIR = 2603).

See Frank et al. (2013) for a description of the method.

Citation: Frank, K.A., Maroulis, S., Duong, M., and Kelcey, B. (2013).
What would it take to change an inference?
Using Rubin's causal model to interpret the robustness of causal inferences.
Education, Evaluation and Policy Analysis, 35 437-460.

Accuracy of results increases with the number of decimals reported.
Code 
# IT index
pkonfound(
  est_eff = -0.63613,
  std_err = 0.16460,
  n_obs = 5282,
  n_covariates = 1,
  index = "IT"
)
Impact Threshold for a Confounding Variable (ITCV):
The minimum (in absolute value) impact of an omitted variable to nullify an inference
for a null hypothesis of an effect of 0 (nu) is based on a correlation of -0.164 with the
outcome and 0.164 with the predictor of interest (conditioning on all observed covariates
in the model; signs are interchangeable if they are different). This is based on a threshold
effect of -0.027 for statistical significance (alpha = 0.05).

Correspondingly the conditional impact of an omitted variable (as defined in Frank 2000) must be
-0.164 X 0.164 = -0.027 to nullify an inference for a null hypothesis of an effect of 0 (nu).

For calculation of unconditional ITCV using pkonfound(), additionally include
the R2, sdx, and sdy as input.

See Frank (2000) for a description of the method.

Citation:
Frank, K. (2000). Impact of a confounding variable on the inference of a
regression coefficient. Sociological Methods and Research, 29 (2), 147-194

Accuracy of results increases with the number of decimals reported.

The ITCV analysis was originally derived for OLS standard errors. If the
standard errors reported in the table were not based on OLS, some caution
should be used to interpret the ITCV.

## Overview

This appendix example estimates the average treatment effect (ATE) of having an IEP on science self-efficacy (`X1SCIEFF`).     race_white = if_else(X1RACE == "White, non-Hispanic", 1, 0, missing = NA),
```{r}
    race_black = if_else(X1RACE == "Black/African-American, non-Hispanic", 1, 0, missing = NA),
    race_asian = if_else(X1RACE == " Asian, non-Hispanic ", 1, 0, missing = NA),
    race_hispanic = if_else(X1RACE == " Hispanic, race specified ", 1, 0, missing = NA),

    # language / locale
    dual_lang = if_else(X1DUALLANG == "First language is English only", 1, 0, missing = NA),
    locale_rural = if_else(X1LOCALE == "Rural", 1, 0, missing = NA),

    # family background
    divorced = if_else(P1MARSTAT %in% c("Divorced", "Separated"), 1, 0, missing = NA),
    bachelorabove1 = if_else(P1HIDEG1 %in% c("Bachelor's degree", "Master's degree"), 1, 0, missing = NA),
    bachelorabove2 = if_else(P1HIDEG2 %in% c("Bachelor's degree", "Master's degree"), 1, 0, missing = NA),

    # parent-reported conditions
    LD = if_else(P1SLD == "Yes", 1, 0, missing = NA),
    DD = if_else(P1DD == "Yes", 1, 0, missing = NA)
  ) %>%
  mutate(
    # SES: convert ordered quintiles into a simple numeric scale 1–5
    SES = factor(
      X1SESQ5,
      levels = c(
        "First quintile (lowest)",
        "Second quintile",
        "Third quintile",
        "Fourth quintile",
        "Fifth quintile (highest)"
      ),
      labels = c(1, 2, 3, 4, 5)
    ) %>% as.numeric()
  )

summary(analytic)
```

## A1.7 Estimate CBPS weights (ATE)

We use **CBPS** to estimate weights that balance covariates by construction.

Then we normalize the weights so the average weight is 1 (helps with interpretability and avoids tiny/huge totals).

```{r}
#| label: a1-cbps-iptw
#| include: true

cbps_fit <- CBPS(
  IEP ~ female + race_white + race_black +
    dual_lang + locale_rural + SES + divorced +
    bachelorabove1 + bachelorabove2 + LD + DD,
  data = analytic_clean,
  method = "exact",
  ATT = 0
)

analytic_clean$IPTW <- cbps_fit$weights
summary(analytic_clean$IPTW)

# Normalize weights: mean(weight) = 1
analytic_clean$finalWeightATE <- with(analytic_clean, IPTW / mean(IPTW))

with(analytic_clean, by(finalWeightATE, IEP, summary))
```

## A1.8 Covariate balance (before vs after weighting)

This is the gut-check step: if weighting doesn’t improve balance, we don’t trust the ATE estimate.

```{r}
#| label: a1-balance-check
#| include: true

covnames <- c(
  "female", "race_white", "race_black",
  "dual_lang", "locale_rural", "SES", "divorced",
  "bachelorabove1", "bachelorabove2",
  "LD", "DD"
)

balance_before <- bal.stat(
  data = analytic_clean,
  vars = covnames,
  treat.var = "IEP",
  w.all = 1,
  get.ks = FALSE,
  sampw = 1,
  estimand = "ATE",
  multinom = FALSE
)

balance_after <- bal.stat(
  data = analytic_clean,
  vars = covnames,
  treat.var = "IEP",
  w.all = analytic_clean[, "finalWeightATE"],
  get.ks = FALSE,
  sampw = 1,
  estimand = "ATE",
  multinom = FALSE
)

balance_before
balance_after
```
